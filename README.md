# Humanoid Teleoperation Isaac Sim

## Overview
This repository contains an initial prototype of humanoid manipulation in Isaac Sim.

## Contents
- Humanoid USD file used in the simulation
- Demo video showing body-part manipulation using Isaac Sim Physics Inspector

## Work Done
- Imported / loaded humanoid model in Isaac Sim
- Used Physics Inspector to move and test humanoid body parts
- Verified that body parts can be manipulated within the simulation environment

## Current Status
This is an early prototype and currently demonstrates manual body-part manipulation through the Physics Inspector. Full teleoperation control is not yet implemented.

## Next Steps
- Improve articulation/joint setup
- Add scripted or input-based control
- Implement teleoperation workflow
- Test more stable humanoid movement and control
